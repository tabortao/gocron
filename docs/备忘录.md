## 重新注册

gocron-node是项目的核心应用，是通过项目web页面`任务节点-自动注册`的命令行安装的，安装后存放到了系统opt/gocron-node目录中。

- 删除定时任务
- 删除任务节点
- 停止docker
- 在终端中删除gocron-node

```bash
sudo -i # 进入管理员权限
cd ..
cd opt
ls
rm -rf gocron-node # 删除文件夹及对应文件
```
